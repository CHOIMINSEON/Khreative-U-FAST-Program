# 학문후속세대(대학원생) 강의기회 제공 및 미래교수자 육성사업(2024)
* 대학원생에게 강의기회를 제공하는 프로그램에 참여해 학부 학생들에게 군집분석(cluster analysis) 기법 중 하나인 K-Means Clustering에 대한 강의를 진행함
* Python과 QGIS를 활용해 K-Means Clustering 실습 및 시각화를 진행

## Directory Structure

```
FAST/
├── 01_data_loading_and_histogram.py       # 데이터 로드 및 히스토그램 시각화
├── 02_kmeans_age_spending_k4.py           # Age-Spending Score 클러스터링 (K=4)
├── 03_kmeans_income_spending_k5.py        # Income-Spending Score 클러스터링 (K=5)
├── 04_interactive_3d_visualization.py     # 3D 인터랙티브 시각화 (K=6)
├── K_Means_주석.ipynb                     # 원본 Jupyter Notebook
└── README.md
```
